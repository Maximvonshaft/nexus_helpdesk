# Definition of Ready

Before implementation starts:

- [ ] This pack is read.
- [ ] Target branch is created.
- [ ] Current tests pass or known failures are documented.
- [ ] Provider choice is confirmed as SES for V1.
- [ ] Email sending identity/domain is available for staging.
- [ ] Staging recipient test mailbox exists.
- [ ] No production enablement is planned before smoke evidence.
- [ ] Reviewer knows migration scope.
