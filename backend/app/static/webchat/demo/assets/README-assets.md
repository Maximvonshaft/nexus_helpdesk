# Speedaf Demo Visual Assets

This package now uses supplied production visual assets instead of relying only on CSS/inline SVG illustration.

## Integrated assets

- `speedaf-logo-official.png` — supplied Speedaf Express wordmark, transparent background, used in header.
- `hero-logistics-scene.webp` / `.png` — premium hero scene with van, parcels, scanner/PDA, route UI cards.
- `speedaf-ai-bot-avatar.png` / `.webp` — AI support avatar used in Webchat header and message avatars.
- `hero-route-map.webp` / `.png` — route-map background layer.
- `hero-location-pin.webp` / `.png` — location pin layer.
- `favicon.svg`, `favicon.ico`, `favicon-32.png`, `apple-touch-icon.png`, `icon-192.png`, `icon-512.png` — browser/app icon suite.
- Optional future composition assets: `hero-van`, `hero-parcel`, `hero-scanner`.

## Integration rule

Webchat, buttons, message bubbles, trust cards, and interactive controls remain real HTML/CSS components. Only brand/hero/avatar/icon visual assets were replaced by production images.
