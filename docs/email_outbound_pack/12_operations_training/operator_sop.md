# Operator SOP — Email Channel

## Sending an email

1. Open ticket.
2. Confirm customer email is correct.
3. Select Email channel.
4. Review From/To/Subject.
5. Write reply.
6. Click Send.
7. Confirm timeline shows Email queued/sent.

## If Email is disabled

Read the missing reasons:
- customer email missing,
- email account not configured,
- sender not verified,
- dispatch disabled,
- recipient suppressed.

Escalate to support lead/admin with ticket id and missing reason.

## If email bounced

1. Do not retry immediately.
2. Confirm customer's email address.
3. Update customer profile if needed.
4. Ask manager/admin to review suppression if applicable.

## If complaint received

1. Do not resend.
2. Escalate to manager.
3. Confirm whether customer consent/context exists.
4. Keep audit trail.

## If customer replies

Reply should appear in the same ticket timeline. If not:
- Search by customer email.
- Check inbound unresolved queue/report.
- Escalate to support lead.
