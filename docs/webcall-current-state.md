# WebCall current state

Nexus uses one LiveKit media plane for browser customers, SIP callers, AI Agents and human operators. The retired PCM WebSocket proxy and AudioWorklet transport no longer exist.

Repository delivery includes the data model, APIs, browser call surface, routing, provider commands, webhook ingestion, configuration UI, tests and residue gate. Real DID, SIP Trunk, LiveKit credentials and production activation remain external fail-closed operations.
