# Definition of Done

- [ ] Email remains blocked by default.
- [ ] Email becomes ready only when all gates pass.
- [ ] Email outbound metadata is persisted.
- [ ] SES provider send path is implemented.
- [ ] Provider message id is persisted.
- [ ] Delivery/bounce/complaint events are ingested.
- [ ] Inbound replies are linked or safely unresolved.
- [ ] Tests cover success and failure paths.
- [ ] Migration upgrade works.
- [ ] Frontend uses capability API.
- [ ] Runbook is complete.
- [ ] Rollback is documented and tested in staging.
- [ ] Final engineer report is filled.
