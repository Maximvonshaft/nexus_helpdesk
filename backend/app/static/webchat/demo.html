Online support demo

Speedaf customer support

This page simulates a customer website with the Speedaf Support webchat widget installed.

Open the chat in the bottom-right corner and send a message. Speedy will reply inside the widget.

If WEBCHAT_VOICE_ENABLED=true, a separate orange voice button appears. It creates a mock WebChat Voice session and opens the dedicated voice page. It does not request microphone access or load WebRTC in this phase.

Snippet:

<script src="/webchat/widget.js" data-tenant="demo" data-channel="website" data-title="Speedaf Support" data-subtitle="Usually replies instantly" data-assistant-name="Speedy" async></script>
<script src="/webchat/voice-entry.js" data-tenant="demo" data-channel="website" data-title="Speedaf Voice Call" data-voice-label="Voice call" async></script>
