# Known Issues and Workarounds

| Issue | Workaround |
|---|---|
| Email not ready because customer email missing | Add verified customer email. |
| Sender identity not verified | Complete provider/domain verification. |
| Bounce event blocks future sends | Confirm/update email; manager review for suppression. |
| Inbound reply not linked | Search by customer email and inspect unresolved inbound records. |
| Provider outage | Use alternative support channel; pause Email rollout. |
| Attachments not supported in V1 | Send text reply and use approved external attachment process until V1.1. |
