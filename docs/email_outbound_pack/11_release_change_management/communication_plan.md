# Communication Plan

## Before rollout

Notify support team:
- Email is being added as a controlled support channel.
- It may initially be available only for selected tickets/markets.
- Disabled reasons are expected if customer email/account is missing.

## During rollout

Post status:
- Feature enabled/disabled.
- Monitored market/team.
- Known limitations.
- Escalation contact.

## After rollout

Share:
- success rate,
- bounce/complaint count,
- inbound linking status,
- next phase.
