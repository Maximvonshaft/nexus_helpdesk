# styles layer

The `styles` layer will own frontend-wide design tokens, base styles, theme declarations, and global style composition.

Target files:

- `tokens.css`
- `base.css`
- `themes.css`
- `globals.css`

Current `webapp/src/styles.css` remains the active stylesheet until a later design-system PR migrates styles safely.

This foundation branch intentionally does not change active CSS imports or visual behavior.
